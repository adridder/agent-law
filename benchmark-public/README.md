# Public Benchmark

This folder is the public home for the agent-law benchmark referenced in the book.

It is intentionally separate from the canonical `LAW.md` and `SOUL.md` texts so the benchmark can evolve without changing the law itself.

When benchmark materials are added here, keep them organized around:
- benchmark prompts and instructions
- rubrics and scoring criteria
- harness code or reproducibility notes
- results, annotations, and score summaries
- only the model-output excerpts needed to document benchmark behavior

License and reuse notes for this directory live in `LICENSE.md` and `NOTICE.md`.
