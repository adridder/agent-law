# Third-Party Sources

This folder contains the source index and the archived reference material used by the research portion of the repository.

## Why this exists

The research should be inspectable. This folder gives readers a place to check where the comparisons, claims, and vendor summaries came from.

## What is here

- [`INDEX.md`](./INDEX.md): the main map of source material, organized by company
- archived files mirrored in this directory when they were useful to keep with the project
- [`NOTICE.md`](./NOTICE.md): the legal and reuse warning for this folder

## How to use this folder

1. Start with [`INDEX.md`](./INDEX.md).
2. Use the listed URLs when you want the original upstream source.
3. Treat mirrored files here as convenience copies, not as evidence that the repository owns or relicenses them.

## Important caution

This directory mixes two things:

- files bundled locally in the repository
- references to upstream materials that are not necessarily mirrored here

If a filename is mentioned in the index but does not appear locally, treat it as an external reference target.
