# Research Materials

This folder contains optional supporting material. It is intentionally separate from the canonical `LAW.md` and `SOUL.md` texts.

Contents:
- `model-behavior-comparison/`: comparison artifacts for model behavior and system-prompt analysis.
- `shepherding-framework/`: builder-facing background principles and the coverage matrix.
- `third-party-sources/`: archived source material and an index of external references.
